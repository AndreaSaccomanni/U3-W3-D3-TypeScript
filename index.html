TypeScript:

Cos'è typescript?

TypeScript è un linguaggio di programmazione open-source sviluppato da Microsoft. Estende le funzonalità di java e perrmette di scrivere un linguaggio più pulito e senza errori, grazie a l'editor che fa un controllo più approfondito, evidenziando incongruenze Converte il codice scruitto in JS ottimizzato e permette di specificare i data types,

Il compilatore TS perché è necessario? e come si usa?

Il compilatore TypeScript è necessario per convertire il codice TypeScript in JavaScript eseguibile. Fornisce un controllo di tipo e errori, ottimizzando il codice e rendendolo più robusto e stabile. Per essere utilizzato va installato globalmente typescript con il comando npm install --g typescript, poi si deve creare un file.ts e eseguendo nel terminale il comando tsc nomefile.ts. Automaticamente si creerà un file .js nella stessa cartella. Eseguendo il comando tsc --init si crea un file di configurazione con le impostazioni predefinite.

Cos'è la Type Inference?

E' una caratteristica che permette al compilatore di dedurre automaticamente il tipo di una variabile, funzione o espressione, senza bisogno di specificarlo esplicitamente. TypeScript è in grado di inferire il tipo giusto in base al valore assegnato o al contesto in cui la variabile è utilizzata.

Type "any":

Quando una variabile è di tipo "any" TypeScript non effettua controlli su di essa, potrebbe assumere qualsiasi valore senza che vengano generati errori. Andrebbe modificato il prima possibile per evitare errori

Type Union:

E' l'unione di due o più tipi assegnabili ad un contenitore di valore. La sintassi è la seguente "let variable: string | number" e variable può essere o di tipo stirnga o di tipo number a seconda del contesto in cui viene utilizzato. Il contesto va specificato con delle condizioni in modo che capisce come deve comportarsi

Le interfacce in TypeScript:

Sono contenitori di tipi con funzionalità di estensione. Un'interfaccia è la struttutra di un oggetto, con le varie caratteristiche, ma contiene i tipi invece che i valori. un'interfaccia quindi ha la possibilità di estenderne un'altra, senza il bisogno di riscrivere tutte le proprietà, ma basta aggiungere le proprietà nuove che verranno integrate automaticamente con quelle presenti nell'interfaccia precedente

I generics:

I generics permettono di creare funzioni, classi, interfacce e tipi che possono lavorare con diversi tipi, ma senza compromettere la sicurezza dei tipi. Utilizzando i generics, puoi scrivere codice riutilizzabile e flessibile, che funziona con qualsiasi tipo, pur mantenendo il controllo rigoroso sui tipi stessi. Un generic type viene dichiarato utilizzando una sintassi di tipo generico, in cui si specifica un parametro di tipo che verrà utilizzato all'interno della funzione, classe o interfaccia. Questo parametro di tipo è generalmente rappresentato da una lettera maiuscola, come T
